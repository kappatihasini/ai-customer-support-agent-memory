# AI Customer Support Agent with Memory

## 🚀 Overview
An AI-powered customer support system that remembers user interactions and provides context-aware, personalized responses.

## 💡 Features
- Persistent memory
- Context-aware replies
- Faster issue resolution
- Adaptive learning

## 🛠 Tech Stack
- Frontend: React / Vite
- Backend: Node.js
- AI: LLM
- Database: Memory storage

## 📂 Project Structure
- ai-support-frontend/
- ai-support-backend/

## 🎯 Impact
- Eliminates repetition
- Improves user experience
- Smart AI-based support
